# 欢迎来到 ex2D 中文文档

ex2D 是一款在 Unity3D 引擎中提供快速 2D 渲染和游戏制作的工具套件。
这款插件包括一个基于 Unity3D Mesh 类编写的 2D 渲染框架，和为这个渲染框架精心设计的制作工具。
这款套件为移动设备上的开发做了及其高效的优化。

本文档将带你深入浅出地学习如何使用 ex2D 在 Unity3D 中制作 2D 游戏。
你还可以在文档中找到 ex2D Runtime 脚本的参考手册。
